### **Installation**:

<br>

```

> Clone the reposity: git clone https://github.com/hstravinskas/Cypress-automation-framework.git

```

```

> npm install

```

<br>

---

### **How to run the tests**:

To run tests:
```

> npx cypress open

```
To run tests in headless mode:
```

> npx cypress run

```
